AVC-Init: Aggregated Visual Constraints for Decoupled Visual-Inertial Initialization

Yangbing Ge1
VCaCLab, Institution Name
Code arXiv

Abstract

高效且鲁棒的初始化是保障视觉惯性里程计 (VIO) 系统性能的基石 。本文提出了一种名为 AVC-Init 的基于聚合视觉约束的解耦式 VIO 初始化框架 。该框架利用任意共视关键帧间的聚合旋转约束，结合预积分合并技术，实现了对陀螺仪零偏的高效独立估计 [cite: 4, 49]。随后，通过全局位置优化策略，在紧耦合与松耦合形式下实现了对速度、重力向量和加速度计零偏的鲁棒估计 [cite: 4, 50]。在最终优化阶段，提出的聚合对极调整 (AEA) 方法在低维空间内完成了全参数的高效联合估计，且无需显式维护点云 [cite: 4, 50]。

Experimental Results

在 EuRoC 公开数据集和仿真环境上的实验表明，AVC-Init 在收敛速度与估计精度方面表现优异 [cite: 5, 265]。相比于基线方法，在存在较大像素噪声的情况下，长基线聚合约束能显著提升陀螺仪零偏的估计鲁棒性 [cite: 264]。

图 3: 不同像素噪声水平下 $b_g$ 估计精度对比，展示了本文方法在噪声环境下的优越性 [cite: 257]。
